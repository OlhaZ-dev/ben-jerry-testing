# Ben & Jerry's UK Website QA Reports
This repository is dedicated to documenting the findings from functional and usability testing of the [Ben & Jerry's UK website](https://www.benjerry.co.uk).

This project is part of non-commercial exploratory testing conducted for educational and portfolio purposes.  
## Project Structure
- `README.md` — project overview (this file)  
- `bug-reports/` — markdown files with detailed step-by-step reports on defects discovered. 
- `reports/` — general summaries and conclusions regarding usability, design, and performance.
- `user-cases/` — markdown files with documented user scenarios (test cases) reflecting expected behavior.
