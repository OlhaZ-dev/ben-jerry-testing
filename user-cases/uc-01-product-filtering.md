# [UC-01] - User Case: Using Search/Filter to Find Product Nutrition Data

**Date Tested:** 2025-04-08

**Feature Focus:** Product Filtering

**Test Type:** Functional

---

## Scenario Details

**Goal:** The user wants to find the **Ingredients and Nutrition Facts** for a specific Ben & Jerry's product. The user should use the search bar or the filter tool on the Flavours page to locate the ingredients information about a specific ice cream product and document the steps taken

**Precondition:**
* The user is on the [Ben & Jerryâ€™s UK Home Page](https://www.benjerry.co.uk/).
* The top navigation menu, including the "Ice Cream Flavours" dropdown, is fully loaded and functional.
* User's device is a desktop

## Steps

| Step # | Action |
| :--- | :--- |
|1.| The user clicks on the **"Ice Cream Flavours"** link in the top navigation menu;
|2.| From the dropdown, the user selects **"Our Flavours"**;
|3.| On the **"Our Flavours"** page, the user taps on the **"Browse by Category"** filter;
|4.| The user selects the **"Ice Cream Tubs"** category;
|5.| From the displayed products, the user clicks on the **"Brookies & Cream"** flavour;
|6. |On the product page, the user scrolls down to the **"About this Flavour"** section;
|7. |Within this section, the user opens the **"Ingredients"** dropdown to view the full list of ingredients and nutritional information.
