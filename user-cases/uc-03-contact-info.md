# [UC-03] - User Case: Locating Company Contact Information

**Date Tested:** 2025-04-08

**Feature Focus:** Navigation / Static Content

**Test Type:** Functional / Usability

---

## Scenario Details

**Goal:** The user wants to find the **contact information** for Ben & Jerry's. The goal is for the user to explore the site, locate the contact details, and document the specific steps taken to find this information.

**Precondition:**
* The user is on the [Ben & Jerryâ€™s UK Home Page](https://www.benjerry.co.uk/).
* The site is fully rendered, and there are no active modal pop-ups (e.g., cookie banner, newsletter sign-up) obscuring the footer area.
* User's device is a desktop

## Steps

| Step # | Action |
| :--- | :--- |
|1.| On the **Home page**, the user scrolls down to the bottom menu panel (footer);
|2.| Under the **"More info"** label he user clicks on the **"Contact Us"** link in the footer;
|3.| On the **"Contact Us"** page, the user clicks on the **"Call Us"** tab;
|4.| The user reviews the phone number and related contact details displayed under the **"Call Us"** section;
|5.| The user clicks on the **"Write to Us"** tab;
|6.| The user reviews the mailing address and customer contact form under the **"Write to Us"** section.
