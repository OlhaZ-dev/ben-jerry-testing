# [UC-05] - User Case: Locating Company Mission and Values

**Date Tested:** 2025-04-08

**Feature Focus:** Static Content

**Test Type:** Usability / Content Verification

---

## Scenario Details

**Goal:** The user wants to find information about the **core values and activism** of the Ben & Jerry's company. The goal is to document the steps the user takes to locate and verify the main content related to the company's social and mission-driven initiatives.

**Precondition:**
* The user is on the [Ben & Jerry’s UK Home Page](https://www.benjerry.co.uk/).

## Steps

| Step # | Action |
| :--- | :--- |
| 1. | On the Home page, the user clicks on the **"Activism"** section in the top navigation menu. |
| 2. | From the dropdown list, the user selects the **"Our Progressive Values"** subsection. |
| 3. | On the **"Our Progressive Values"** page, the user scrolls through the content and explores detailed information about Ben & Jerry’s values. |
