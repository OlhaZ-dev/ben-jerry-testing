# [UC-02] - User Case: Historical Data Retrieval

**Date Tested:** 2025-04-08

**Feature Focus:** Static Content

**Test Type:** Usability

---

## Scenario Details

**Goal:** The user wants to learn more about the **company's history**. The goal is for the user to find the year the first Ben & Jerry's ice cream shop was opened. The user must list the steps taken to find this information.

**Precondition:**
* The user is on the [Ben & Jerry’s UK Home Page](https://www.benjerry.co.uk/).
* There are no active site overlays (e.g., cookie banners or newsletter sign-up modals) that would block access to the navigation menu.
* User's device is a desktop

## Steps

| Step # | Action |
| :--- | :--- |
|1. |The user clicks on the **"About Us"** link in the top navigation menu;
|2.| From the dropdown menu, the user selects the **"Overview"** subsection;
|3. |On the **"Overview"** page, the user scrolls through the content and locates the information about the first Ben&Jerry’s ice cream shop, including the year it was opened.
